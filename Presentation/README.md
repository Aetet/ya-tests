Презентации
=========

Система для показа презентаций. Подумайте, как реализовать переключение слайдов и навигацию. Предусмотрите возможность размещения нескольких презентаций на одной странице.

Установка
------------

`npm i`

Сборка
-----

`node_modules/.bin/component build -d -o web/build`

Запуск
---

Открыть в браузере файл web/index.html
